地域解鎖

GPS 定位 × 真實場景學習
走到哪裡學到哪裡 — 位置就是你的課本

當你靠近特定地點，App 自動解鎖專屬日語內容：發音、情境對話、文化小知識、實拍菜單/標示，甚至慢速導覽語音。
讓街道、餐廳、神社、車站主動「教」你實用日語。

解鎖場景實例

附近餐廳

點餐用語 + 菜單實拍解說

  • すみません、メニューをお願いします Sumimasen, menyū o onegai shimasu. （不好意思，請給我菜單）
  • おすすめは何ですか？ Osusume wa nan desu ka? （有什麼推薦的嗎？）
  • これをお願いします Kore o onegai shimasu. （我要這個）
  • 美味しかったです、ごちそうさまでした Oishikatta desu, gochisōsama deshita. （很好吃，謝謝款待）
  • App 會顯示實拍菜單 + 發音 + 過敏詢問（如エビアレルギーです）

神社 / 寺廟

參拜禮儀 + 相關敬語與文化

  • 二礼二拍手一礼（にれいにはくしゅいちれい） Nirei nihakushu ichirei （二鞠躬、二拍手、一鞠躬）
  • お賽銭をお入れください O-saisen o oire kudasai （請投入香油錢）
  • ご利益がありますように Goryaku ga arimasu yō ni （祈求靈驗）
  • お参りに行きます O-mairi ni ikimasu （去參拜）

App 會播放洗手順序語音 + 拍手示範 + 繪馬寫法提示

車站 / 便利店

日常交通 + 購物對話

  • 切符はどこで買えますか？ Kippu wa doko de kaemasu ka? （車票在哪裡買？）
  • ポイントカードはお持ちですか？ Pointo kādo wa omochi desu ka? （有集點卡嗎？）
  • 温めますか？ Atatamemasu ka? （便當要加熱嗎？）
  • 袋はいりますか？ Fukuro wa irimasu ka? （需要袋子嗎？）
  • App 即時顯示附近車站路線 + 便利店熱門商品發音（如おにぎり、コーヒー）

推介熱門旅遊景點 × 語言沉浸

這些日本超人氣景點不只美，還超適合練習日語！
當你走到這裡，App 會自動解鎖更多在地詞彙、對話、文化解說，讓旅行變成最有效的「沉浸式學習」。

東京 (Tokyo)

現代都市 + 日常對話天堂

  • 澀谷交叉口、新宿、淺草寺、晴空塔
  • 練習購物、問路、點餐、搭電車的超實用日語
  • 人多互動機會多，適合初學者快速開口

京都 (Kyoto)

古都文化 + 敬語/傳統練習

  • 清水寺、伏見稻荷大社、金閣寺、嵐山竹林、祇園
  • 參拜、穿和服、茶道、寺廟導覽 → 學敬語與文化用語
  • 寧靜氛圍，適合深度沉浸與慢速練習

大阪 (Osaka)

關西方言 + 美食街頭聊天

  • 道頓堀、心齋橋、大阪城、環球影城
  • 大阪人熱情愛聊天 → 練習搞笑、點章魚燒、問路超自然
  • 美食天堂，邊吃邊學最開心

廣島 (Hiroshima) + 宮島

和平文化 + 嚴肅話題練習

  • 和平紀念公園、嚴島神社、廣島燒
  • 學歷史相關詞彙、感謝語、祈福用語
  • 情感深度對話機會多

箱根 (Hakone)

溫泉 + 自然景觀描述

  • 蘆之湖、箱根纜車、大涌谷、溫泉旅館
  • 練習天氣、風景、泡湯相關日語
  • 放鬆環境，適合中級以上練習敘述句

北海道 (Hokkaido)

自然 + 四季活動用語

  • 札幌雪祭、富良野花田、小樽運河、登別地獄谷
  • 學季節詞彙、海鮮點餐、滑雪/賞花相關對話
  • 鄉村地區，當地人互動更親切
